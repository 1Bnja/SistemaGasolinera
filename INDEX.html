Universidad de Talca - Proyecto 2 Sistemas Distribuidos

Instrucciones de Uso

1. Asegúrate de que Docker Compose esté ejecutándose: docker-compose up --build
2. Haz clic en cualquier enlace para acceder a los componentes del sistema
3. Navega entre componentes usando los botones de navegación en cada interfaz
